## Instalacion

1. Clonar el proyecto

```bash
   git clone https://github.com/Hernan-99/Challenge-Uteam-Frontend.git
```

2. Moverse a la carpeta del proyecto

```bash
cd Challenge-Uteam-Frontend
```

3. Instalar dependencias

```bash
npm install
```

3. Iniciar servidor

```bash
npm run dev
```
